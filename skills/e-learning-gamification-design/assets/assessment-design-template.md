## Scoring Guide

### Point Allocation

| Question Type | Points Each | Total Points |
|---------------|-------------|--------------|
| Multiple Choice | 5 | [#] |
| Multiple Response | 10 | [#] |
| Matching | 15 | [#] |
| Scenario | 20 | [#] |
| **Total** | | [#] |

### Grade Scale

| Percentage | Grade | Status |
|------------|-------|--------|
| 90-100% | A | Pass with Distinction |
| 80-89% | B | Pass |
| 70-79% | C | Pass |
| 60-69% | D | Marginal Pass |
| Below 60% | F | Fail |

### Partial Credit Rules

| Question Type | Partial Credit Rule |
|---------------|---------------------|
| Multiple Response | +2 points per correct selection, -1 per incorrect |
| Matching | +3 points per correct match |
| Sequencing | +2 points per item in correct position |
| Scenario | No partial credit |

## Feedback Strategy

### Immediate Feedback (Formative)

| Situation | Feedback Type | Timing |
|-----------|---------------|--------|
| Correct answer | Positive reinforcement + explanation | Immediately |
| Incorrect answer | Corrective guidance + explanation | Immediately |
| Partial credit | Detailed breakdown | Immediately |
| Hint requested | Scaffolded support | On request |

### Delayed Feedback (Summative)

| Situation | Feedback Type | Timing |
|-----------|---------------|--------|
| Assessment complete | Score summary | After submission |
| Review mode | All questions with answers | After score display |
| Detailed report | Performance by objective | On demand |

### Feedback Message Templates

**Correct Answer:**
```
✓ Correct!

[Explanation of why this is correct]

Key point: [Main takeaway]
```

**Incorrect Answer:**
```
✗ Not quite right.

[Explanation of why the selected answer is incorrect]

The correct answer is [Letter] because [reason].

Remember: [Key concept to remember]
```

**Partial Credit:**
```
◐ Partially correct

You got [#] out of [#] correct.

[What you got right]
[What you missed and why]

Review: [Link to relevant content]
```

## Pilot Testing Plan

### Pilot Test Protocol

| Step | Action | Duration |
|------|--------|----------|
| 1 | Recruit pilot testers | [Timeframe] |
| 2 | Administer assessment | [Timeframe] |
| 3 | Collect feedback | [Timeframe] |
| 4 | Analyze results | [Timeframe] |
| 5 | Revise assessment | [Timeframe] |

### Item Analysis Metrics

| Metric | Target | Action if Not Met |
|--------|--------|-------------------|
| Difficulty Index | 0.4-0.7 | Revise too easy/hard items |
| Discrimination Index | >0.3 | Review or replace item |
| Distractor Effectiveness | Each >5% selection | Replace ineffective distractors |
| Point-Biserial | >0.2 | Review item quality |

### Pilot Feedback Form

## Quality Assurance Checklist

### Content Review

- [ ] All questions align with learning objectives
- [ ] Questions are free of bias and stereotypes
- [ ] Language is clear and appropriate
- [ ] Correct answers are actually correct
- [ ] Distractors are plausible but clearly wrong
- [ ] Feedback is helpful and accurate

### Technical Review

- [ ] All questions render correctly
- [ ] Scoring works as expected
- [ ] Feedback displays correctly
- [ ] Navigation functions properly
- [ ] Timer works correctly (if applicable)
- [ ] Results are saved and reported correctly

### Accessibility Review

- [ ] Screen reader compatible
- [ ] Keyboard navigable
- [ ] Color contrast sufficient
- [ ] Alt text provided
- [ ] Extended time available
